
# Design challenge

This is mini-project which purpose was to use a figma desgin and convert it to re-usable design system and react components.


## Tech Stack

**Client:** React, TypeScript, Sass, StoryBook  
**Tools:** Vite 





## How to run

Inside the project root folder

```bash
  npm install
  npm start
```
